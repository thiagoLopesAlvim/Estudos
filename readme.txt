para rodar o programa em outra maquina é necessario:

1-Instalar o PHP

2-Instalar o Composer

3-Instalar o Vite

3.1-Instalar Nodejs

4-Driver do SQL Server

5-Adicionar ao gpedit.msc os Script de inicialização

6-Abrir o Link do programa -http://127.0.0.1:8000/


Plugins a serem instalados no composer com o comando "composer require":

composer require igorescobar/jquery-mask-plugin

composer global require laravel/installer

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
